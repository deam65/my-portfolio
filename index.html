I like to code! Here are some projects I'm proud of:
Multithreaded HTTP Server

Developed an HTTP server that responded to GET, PUT, and HEAD requests, using POSIX threads to ensure that simultaneous requests could be processed with no busy waiting. Developed in C, communicating locally with curl/netcat.

Metastatic Region Analysis

Collaborated with peers to develop a convoluted neural network (CNN) that could identify metastatic regions in lymph node tissue cells relatively accurately (up to 87%).

FiTrak

Collaborated with peers to develop a Web + Android fitness application, aimed at connecting trainers with trainees. Project was developed in a Scrum environment, using the Ruby on Rails framework and Firebase Realtime.
